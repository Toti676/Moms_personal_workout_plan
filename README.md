# Moms_personal_workout_plan
A personal single html file that can be downloaded and loaded straight into any browser which then shows a personal full week workout plan which I have designed for my dearest Mother.  I made this project to combine my love of working out and my love of coding to help my Mother live a healthier life. 

Check it out:
- To check out my project just download the Moms-workout-plan.html file to your device and run it on any web browser.

Disclaimer:
- Not supported for Iphones. Would need to download a HTML viewer from the app store. (I recommend HTML viewer and watch a short youtube video on how to work it)
